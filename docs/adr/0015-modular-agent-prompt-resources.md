# Modular agent prompt resources

The Agent prompt will be assembled from ordered Markdown resources for base behavior, personality, execution mode, approval policy, context management, and handoff guidance. This preserves Java prompt-layering behavior while allowing each Python-specific capability and tool contract to remain accurate and independently testable, rather than maintaining one monolithic source string. Stable tool-selection guidance belongs in the base prompt, while a runtime section may summarize currently available tools; complete tool schemas remain exclusively in the API `tools` field. The runtime section is rebuilt before every model turn, so model switches, approval-policy changes, and current MCP tool availability apply on the next request without entering session history. The default interaction language is Chinese, while code, commands, paths, API names, and user-requested foreign-language text remain unchanged.
